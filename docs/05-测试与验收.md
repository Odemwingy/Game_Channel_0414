# 测试与验收

## 1. 测试范围

- 功能链路：注册、游玩、记分、导出
- 联机链路：建房、入房、对局、结算、重连
- 会员链路：绑定、导出、航后同步
- 运维链路：航班初始化、结束、重置

## 2. 验收标准（核心）

- MVP 主链路完整可用，无阻断缺陷。
- 积分对账一致：汇总分与流水分完全一致。
- 联机重连恢复：网络恢复后 3 秒内完成 `sync_full`。
- 导出可复算：同一批次重放结果一致。

## 3. 测试矩阵

| 类别 | 用例 | 预期 |
|---|---|---|
| 功能 | 单机一局结束 | 生成会话与积分流水 |
| 功能 | 会员绑定 | 保存哈希并发放奖励 |
| 联机 | 两人对局全流程 | 状态机按序流转 |
| 联机 | 掉线120秒内重连 | 恢复席位并全量同步 |
| 安全 | 非法动作提交 | 返回 `game:error` 并审计 |
| 可靠 | 航班导出重试 | 保持同一 `batch_id` 幂等 |
