# 外部游戏团队交付清单

- 联机协议与状态机文档（模块 C）
- 最小客户端 SDK（TypeScript）
- 服务端插件模板（新增联机游戏）
- 本地联调与验收清单

## 2. 交付物目录

- 接入总览：`docs/09-联机游戏接入指南.md`

- 服务运行说明：`server/README.md`

## 3. SDK 能力边界

`MultiplayerClient` 当前覆盖：
- 连接管理：`connect/disconnect`
- 房间流程：`createRoom/joinRoom/leaveRoom/reconnectRoom/requestFullSync`
- 对局流程：`ready/sendAction`
- 事件订阅：`room:sync`、`game:start`、`game:update`、`game:sync_full`、`game:over`、`game:error`

说明：
- SDK 为最小实现，适合接入 PoC / MVP。
- 生产环境建议在业务层补充重试、埋点、断线恢复 UI 与动作队列。

## 4. 对外联调基线

1. 启动服务：`cd server && npm install && npm run dev`
2. 按 `docs/09-联机游戏接入指南.md` 完成建房/入房/准备/对局
3. 验证幂等与恢复：
   - 重复 `room:join` 不清空 `ready`
   - 重复 `game:action(actionId)` 不重复执行
   - 断线后可通过 `room:reconnect` + `game:sync_full` 恢复
4. 对照验收项：`docs/09-联机游戏接入指南.md` 第 9 章

## 5. 版本与变更约定

- 协议版本：`/api/v1` + 当前 WebSocket topics
- 兼容策略：新增字段仅追加，客户端需忽略未知字段
- 变更流程：破坏性变更必须升级大版本并提供迁移说明
